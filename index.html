This project is a custom implementation of core C++ STL containers using Object-Oriented Programming principles. It covers custom-built vector, map, set, stack, and pair classes, along with a complete Red-Black Tree to support balanced associative data structures. You'll get an in-depth understanding of how these containers work under the hood.

‚ú® Key Highlights & Innovations

  • Custom Vector: Manual memory management, iterators, dynamic resizing using raw pointers.
  • Custom Map: Implemented using a fully functional Red-Black Tree for key-value storage.
  • Custom Set: Built on Red-Black Tree to manage unique sorted elements efficiently.
  • Custom Stack: LIFO structure using linked list, with push/pop/top methods and O(1) operations.
  • Custom Pair: Templated class to mimic `std::pair` supporting any two types.
  • Red-Black Tree: Full logic with insertions, deletions, rotations, color flipping, and balancing rules.
  • Templates: All containers are generic and support reusable type-safe code.
  • Object-Oriented Programming: Classes, constructors, destructors, operator overloading, and encapsulation.
  • Educational Goal: Learn STL internals and strengthen data structure & OOP understanding.

üß† Behind the Code

All data structures were written using only core C++ concepts without relying on built-in STL containers. Each file showcases deep insight into internal implementations.

The Red-Black Tree manages balance via color rules and rotations. Vector mimics real capacity doubling and buffer copying. Set and map derive directly from the RBT structure. Stack operates as a clean linked list, and pair uses template specialization.

This project enhances your understanding of container behaviors, algorithmic complexity, and object design ‚Äî making it ideal for learning, teaching, and showcasing skills in interviews or contributions.
